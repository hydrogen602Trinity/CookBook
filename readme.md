# How to Run

`python run.py`

This command is compatible with both python 2 and 3. It will look for python3 itself.

## Installing dependencies
`python run.py install`

## Testing
`python run.py test`

## Running
Open two terminals:
Run `python run.py run` in one
Run `python run.py run-front` in the other
